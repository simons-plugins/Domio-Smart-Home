# Support

## Before opening an issue

Please check whether your issue is with this app or with your Indigo server:

- **App issues** (UI bugs, crashes, connection problems from the app side): Open an issue here
- **Indigo server/plugin issues** (devices not responding, automation problems, plugin errors): Please visit the [Indigo Domotics Community Forums](https://forums.indigodomo.com/)

## Reporting an app issue

Please include:
- iOS version
- App version (found in Settings > Support & About)
- Indigo server version
- Steps to reproduce the issue
- Screenshots if applicable
