# Home Remote for Indigo

A modern iOS app for controlling [Indigo](https://www.indigodomo.com/) home automation systems.

## About

**Home Remote for Indigo** is an independent, community-built app. It is **not made by or affiliated with Indigo Domotics.**

Features:
- Real-time device control (switches, lights, dimmers, thermostats)
- Action groups (scenes) execution
- Event log viewer
- Voice assistant integration
- Secure remote access via Indigo Reflector

## Support

- **App issues** (bugs, crashes, feature requests): [Open an issue here](https://github.com/simons-plugins/Home-Remote-for-Indigo/issues)
- **Indigo server/plugin issues** (devices not responding, automation problems, plugin errors): Please visit the [Indigo Domotics Community Forums](https://forums.indigodomo.com/)
